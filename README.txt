# My Python Project

## Setup

1. Clone the repo:
   ```bash
   git clone https://github.com/username/my_project.git
   cd my_project

2. Create and activate a virtual environment:
python -m venv .venv
# Linux/macOS
source .venv/bin/activate
# Windows
.venv\Scripts\activate

3. Install dependencies:
pip install -r requirements.txt

4. Run the project:
python test.py
